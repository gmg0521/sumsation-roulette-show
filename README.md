# 🎡 천국의 룰렛 소개 Show (광고 없는 모던 룰렛)

수련회 및 레크리에이션 기획을 위해 광고 없이 깔끔하고 고급스러운 디자인으로 구현한 룰렛 웹 애플리케이션입니다.
순수 HTML, CSS, JavaScript로 구현되어 있어 별도의 서버 빌드 없이 정적으로 아주 빠르게 동작합니다.

## 🚀 GitHub Pages를 통한 무료 배포 방법

본 프로젝트에는 GitHub Actions를 이용한 자동 배포 워크플로우가 구성되어 있어, 깃허브 리포지토리에 푸시하는 것만으로 간단히 웹페이지로 실행할 수 있습니다.

### 배포 단계

1. **GitHub 리포지토리 생성 및 업로드**
   - GitHub에 새로운 public 리포지토리를 생성합니다.
   - 로컬 프로젝트 폴더 안의 모든 파일(`.github` 폴더 포함)을 해당 리포지토리의 `master` 브랜치에 푸시(Push)합니다.

2. **GitHub Pages 설정 변경**
   - 깃허브 리포지토리 페이지로 이동한 뒤 **Settings** > **Pages** 탭을 클릭합니다.
   - **Build and deployment** 영역의 **Source** 설정을 `Deploy from a branch`에서 `GitHub Actions`로 변경합니다.

3. **자동 배포 확인**
   - 변경 후 상단의 **Actions** 탭으로 이동하면 `Deploy static content to Pages` 워크플로우가 자동으로 실행되는 것을 볼 수 있습니다.
   - 빌드 및 배포가 완료되면 Settings > Pages 탭 상단에 나타나는 주소(예: `https://<사용자이름>.github.io/<리포지토리이름>/`)로 접속하여 룰렛을 바로 실행할 수 있습니다.

## 🎨 주요 기능

- **실시간 리스트 연동**: 우측의 참가자 명단 텍스트 창을 편집하면 실시간으로 왼쪽 룰렛 판의 칸과 이름이 갱신됩니다.
- **20가지 수련회 주제 프리셋**: 우측 드롭다운에서 주제를 선택하면 즉시 20개 질문 카드(취미, 여행, ..., 동물, 자유주제) 목록이 룰렛에 채워집니다.
- **순서 섞기 (Shuffle) & 비우기 (Clear)**: 참가자들의 순서를 무작위로 섞거나 전체 명단을 삭제합니다.
- **긴 이름 말줄임 처리**: 이름이 룰렛 크기보다 길어지면 중앙 영역을 침범하지 않도록 말줄임표(`...`)로 자동 축소 처리합니다.
- **오디오 & 시각 이펙트**: 
  - 스핀 시 속도와 바늘 튕김 궤적에 완벽히 싱크된 째깍거리는 물리 효과음 합성.
  - 당첨 시 영롱한 실로폰 축하 멜로디 및 종이가루(Confetti) 폭죽 세레모니.
